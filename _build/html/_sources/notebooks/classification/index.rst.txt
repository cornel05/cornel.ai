=================
Classification
=================

Classification is about teaching machines to recognize categories. From spam detection to medical diagnosis, most real-world decisions are about **choosing among discrete options**. This chapter introduces the core methods for turning raw data into well-defined class boundaries, evaluating model performance, and reasoning about uncertainty in predictions.
